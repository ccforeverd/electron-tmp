# ElectronTmp

a template for electron vue.

## Project Setup

``` bash
yarn config set electron_mirror https://npm.taobao.org/mirrors/electron/ -g

yarn
```

### Development

- terminal(1): start webpack

  ``` bash
  yarn serve
  ```

- terminal(2): start electron

  ``` bash
  yarn start
  ```

### Production

``` bash
yarn build
```

### Unit Tests

``` bash
yarn test:unit
```

### Lints and fixes files

``` bash
yarn lint
```
